## 核心流程

### Step1:自动化浏览器操作（**目标：登录系统并导出原始数据**）

- 1.**启动与导航**：打开Chrome浏览器，访问网址wuye-zy.ztstc.cn。
- 2.**登录逻辑**：检测登录页面的账号密码输入框。如果输入框中已存在数据（由浏览器自动填充），则直接点击登录按钮。如果未填充，请尝试触发浏览器的密码保存功能或提示我。
- 3.**菜单导航**：成功登录后，按顺序点击左侧导航菜单：首先点击“收费管理”，然后在出现的二级菜单中点击“费用生成”，接着在页面上找到并点击“常规费用一览表”进入查询界面。
- 4.**筛选条件**：

    - 4.1 **组织机构**：点击“组织机构”下拉多选框，全选其中所有选项。
    - 4.2 **收费项目**：点击“收费项目”下拉框，选中“物业费”这一项。
    - 4.3 **应收日期**：判断机制：首先判断用户的指令中是否明确指定了具体的日期范围。

        - **情况A（未指定日期）**：若无指定，则自动执行默认逻辑——开始日期设为本年度1月1日，结束日期设为当前系统时间所在月份的最后一天（本月底），日期格式为：YYYY-MM-DD。

        - **情况B（指定日期）**：若指令中包含具体日期（如“3月份”或“2026-03-01至2026-03-31”），则严格按照指定日期范围进行设置。
- 5.**执行与导出**：点击“搜索”按钮。待页面数据加载完成后，点击“导出”按钮，将数据保存至donwload/。

### Step2:数据处理

- 1.**文件识别**：监控下载文件夹，识别刚刚导出的最新文件，将其作为原始数据源。
- 2.**运行脚本**：将导出的原始文件路径作为参数传递给脚本，脚本路径为："scripts/statistics.py",
    - **注意：** 脚本需要接收命令行参数。
    ```bash
    # 统计数据脚本
    python scripts/statistics.py [导出的数据文件]
    ```
    
- 3.**生成报表**：脚本执行完成后，会在当前位置生成最终的周例会报表。


### Step3:反馈
- 1.**反馈信息**：在任务完成后，请向我报告：导出的文件名、脚本执行状态（成功/失败）以及最终报表的保存位置。
